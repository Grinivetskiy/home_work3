Селекторы бывают

  • Универсальный
    Соответствует любому html документу
  • Теговый
    Позволяет создавать стили для всех элементов этого типа
  • Классовый
    Позволяет задавать стили для одного и более элементов с одинаковым именем класса
  • Селектор идентификатора
    Позволяет задавать стили для конкретного элемента
  • Селектор потомка
    Позволяет задавать стили к элементам,расположенных внутри элеменета-контейнера
  • Дочерний
    Позволяет задавать стили к элементам,которые являются прямыми потомками

    • Соседний
    Выбирает элемент, который находиться непосредственно после указанного элемента

    Родственный
    Выбирает все элементы, которые находятся после указанного элемента

    Селектор атрибута
    Позволяет задавать стили к элементам на основе имени атрибута или значении атрибута

    Селектор групп
    Позволяют задавать стили сразу нескольким элементам
